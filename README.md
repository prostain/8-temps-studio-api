# 8-temps-studio-api

## Installation

```sh
$ npm install
```

insérer le fichier config.json dans le dossier db

## Sequelize

```sh
sequelize db:migrate

sequelize-cli db:migrate:undo:all

npx sequelize-cli seed:generate --name user //ne pas tenir compte

npx sequelize-cli db:seed:all
```

## run
```sh
node app.js
```

## Url

localhost:3000
